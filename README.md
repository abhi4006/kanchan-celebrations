# कंचन का मंगल उत्सव

Next.js, Tailwind CSS, shadcn/ui-style components और Motion से बना responsive
वधू पक्ष का तीन-screen हिंदी invitation:

1. श्री गणेश cover card
2. 01 जुलाई 2026 का सगाई समारोह
3. 03 जुलाई 2026 का शुभ विवाह

दोनों समारोहों में live countdown, Google Maps venue links, animations और music
control शामिल हैं। Wedding page पर बारात का event `बारात आगमन` है।

## Local development

```bash
npm install
npm run dev
```

फिर `http://localhost:3000` खोलें।

## Deployment

`main` branch पर push होने के बाद GitHub Actions static export को GitHub Pages
पर deploy करता है:
